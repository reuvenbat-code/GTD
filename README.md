# GTD — ניהול משימות | הוראות התקנה

## קבצים בתיקייה זו
- `index.html` — האפליקציה המלאה
- `manifest.json` — הגדרות PWA
- `sw.js` — Service Worker (מאפשר עבודה offline)
- `icon-192.png` / `icon-512.png` — אייקונים

---

## שלב 1: יצירת Repository ב-GitHub

1. היכנס ל-**github.com** עם המשתמש `Reuvenbat`
2. לחץ על **+** (ימין למעלה) → **New repository**
3. **Repository name**: `gtd` (חשוב! בדיוק ככה)
4. ודא שמסומן **Public**
5. לחץ **Create repository**

---

## שלב 2: העלאת הקבצים

בדף ה-repository החדש:

1. לחץ **uploading an existing file** (הקישור הכחול באמצע הדף)
2. גרור את כל 5 הקבצים מהתיקייה לחלון (או לחץ "choose your files")
3. למטה, לחץ **Commit changes**

---

## שלב 3: הפעלת GitHub Pages

1. ב-repository שלך, לחץ על **Settings** (גלגל שיניים)
2. בתפריט שמאל: לחץ **Pages**
3. תחת **Branch**: בחר `main` ← `/ (root)`
4. לחץ **Save**
5. המתן כ-2 דקות

---

## שלב 4: הוספה למסך הבית (Android)

1. פתח **Chrome** בטלפון
2. היכנס לכתובת: **`https://reuvenbat.github.io/gtd`**
3. לחץ על שלוש הנקודות (⋮) בפינה הימנית העליונה
4. בחר **"הוסף למסך הבית"** או **"Install app"**
5. לחץ **הוסף**

✅ האפליקציה תופיע על מסך הבית כמו אפליקציה רגילה!

## שאלות נפוצות

**האפליקציה עובדת בלי אינטרנט?**
כן. אחרי פתיחה ראשונה עם אינטרנט, עובדת offline מלא.

**הנתונים נשמרים איפה?**
ב-localStorage של הדפדפן בטלפון. לא נמחקים אלא אם תנקה את נתוני Chrome.

**איך לגבות?**
פתח את האפליקציה → סיידבר → "📤 ייצוא JSON" → שמור את הקובץ.
